## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

Navigate into the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
